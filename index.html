üì¶ Installation

Get started with PicBan in minutes

Install from Chrome Web Store

  1. Visit Chrome Web Store
    Click the "Add to Chrome" button below or search for "PicBan" in the Chrome Web Store
  2. Add to Chrome
    Click the "Add to Chrome" button on the extension page
  3. Confirm Installation
    Click "Add extension" in the confirmation dialog
  4. Pin the Extension
    Click the Extensions icon (puzzle piece) in Chrome toolbar and pin PicBan for easy access
  5. Start Using
    Click the PicBan icon to configure your settings and you're ready to go!
